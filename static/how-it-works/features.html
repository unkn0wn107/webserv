Supported operations

TextLab is suggesting operations that can be performed based on your input. For example, if you copy and paste JSON into the application, it suggests to validate and format it.

The following operations are supported:

I. HTML, XML

  • Format
  • Validate & Clean up
  • Validate & Clean up & Format
  • XML to JSON
  • Save as HTML, XLM
  • Save as escaped HTML, XML

II. JSON

  • Validate & Format
  • Clear
  • JSON to JavaScript
  • JSON to XML
  • Save as JSON

III. SQL

  • Validate
  • Save as SQL

IV. URL

  • Encode
  • Decode
  • Extract params

V. CSV

  • Preview
  • Save as CSV
  • Export to Excel
  • Export with other delimiters

VI. Text

  • Uppercase
  • Lowercase
  • Capitalize
  • Camelcase
  • Remove additional white spaces
  • Base64 encoding and decoding
  • Unique values
  • Cryptographic hash functions: MD2, MD5, SHA1, SHA256, SHA384 and SHA512
  • Keep Numbers Only

VII. Numbers

  • Mili-seconds to Date
  • Integer to binary
  • Integer to hex
  • Integer to octal
  • Remove additional white spaces
  • Base64 encoding and decoding

TextLab is available for MAC OS X while it is compatible with OS X 10.7.4 or later (64-bit processor) and you can purchase it in the App Store for the launching price $8.99.


TextLab was originally released on August 19th 2015.
